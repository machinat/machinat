# Create Machinat App

Project creator for Machinat framework.

## Usage

Create a hello-world project with specified platforms:

```bash
# with npm
npm init @machinat/app -- -p <platform> [-p ...] my-project

# with yarn
yarn create @machinat/app -- -p <platform> [-p ...] my-project
```
